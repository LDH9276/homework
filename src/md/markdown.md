# 마크다운 문법

## 1. 마크다운이란?

2004년 존 그루버에 의해 만들어진 언어이다.  
마크다운의 문법을 통해 HTML으로 변환이 가능하며, 특수기호와 문자를 이용하여 표현이 가능하다.
LLM AI (GPT, 제미나이, 클로드)에서 자주 사용되며 특히 최근들어

```
너 정말 **핵심**을 찔렀어
```

같은 밈으로도 쓰고 있다.
또한 개발자들이 모이는 Velog나 Git hub 문서에서도
README.md와 같이 마크다운을 직접적으로 지원하며, 이를 통해 다음과 같이 쓸 수 있다.

## 2. 기본 글 문법

마크다운 문서는 문장 중간에 다음과 같은 서식 이용이 가능하다.

```
*이텔릭체*
_이텔릭체_
**강조**
__강조__
~~취소선~~
```

_이텔릭체_  
_이텔릭체_  
**강조**  
**강조**  
~~취소선~~

다음은 줄 바꿈이다. 공백 3개면 다음줄로 넘어간다.

```
* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.
이렇게

* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.___\\ 띄어쓰기
이렇게
```

- 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.
  이렇게

- 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다. 띄어쓰기
  이렇게

### 마크다운의 헤더

마크다운은 (#)를 이용하여 대제목 - 중제목 - 소재목으로 가능하다.  
즉, HTML으로 보면 H태그와 동일한 기능이다.

```
[사용방법]
# 대제목
## 중제목
### 소제목
```

## 사용예

# 대제목

## 중제목

### 소제목

### 인용문 태그

blockquote로 > 을 통해 인용문을 작성한다.  
작성예

```
[사용방법]
> 인용문 입력
```

> 인용문1
>
> > 인용문2
> >
> > > 인용문3

### 리스트 목록 (숫자, 번호, 기호)

ol, ul, dl과 동일하다

```
[사용방법]
1. 첫번째
2. 두번째
3. 세번째

* 빨강
    * 주황
        * 파랑
```

1. 첫번째
   - 부제1
   - 부제2
   - 부제3

2. 두번째
   - 목록1
   - 목록2
   - 목록3

### 코드

코드는 백틱문자(`)를 입력하며 이를 동해 자유롭게 코드나 마크다운 문법 등  
필요한 내용을 입력이 가능하다.

````
    [사용방법]

    ```
        안에 들어갈 내용
    ```
````

또한 코드 블럭 뒤에 언어를 지정해 실행도 가능하다.

````
    ```javascript
    console.log("Hello");
    ```
````

```javascript
console.log('Hello');
```

### 표

table코드 처럼 표를 만들어 준다.

```
[사용방법]
| 1 | 2 |
|---|---|
| 1 | 2 |
```

| 1   | 2   |
| --- | --- |
| 1   | 2   |

### 체크리스트

체크리스트로 다음과 같이 목차나 해야할 일이나 계획 (예를 들면 앞으로의 수정사항) 등을 목록화도 가능하다.

```
[사용방법]
- [ ] 안 한 일
- [x] 완료한 일
```

- [ ] 안 한 일
- [x] 완료한 일

### 이미지나 링크

이미지는 다음과 같이 사용 가능하다. []

```
[사용방법]
[Google 바로가기](https://google.com)   # 링크로 이동하는 방법
![강아지 사진](/image/dog.jpg)           # 앞의 글은 alt 속성을 가진다.
```
