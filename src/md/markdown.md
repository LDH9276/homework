# 마크다운 문법

# 마크다운이란?

2004년 존 그루버에 의해 만들어진 언어이다.  
마크다운의 문법을 통해 HTML으로 변환이 가능하며, 특수기호와 문자를 이용하여 표현기 가능하다.

# 기본 글 문법

마크다운 문서는 문장 중간에 다음과 같은 서식 이용이 가능하다.

```
*이텔릭체*
_이텔릭체_
**강조**
__강조__
~~취소선~~
```

_이텔릭체_  
_이텔릭체_  
**강조**  
**강조**  
~~취소선~~

다음은 줄 바꿈이다. 공백 3개면 다음줄로 넘어간다.

```
* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.
이렇게

* 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.___\\ 띄어쓰기
이렇게
```

- 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다.
  이렇게

- 줄 바꿈을 하기 위해서는 문장 마지막에서 3칸이상을 띄어쓰기해야 한다. 띄어쓰기
  이렇게

# 마크다운의 헤더

마크다운은 (#)를 이용하여 대제목 - 중제목 - 소재목으로 가능하다.  
즉, HTML으로 보면 H태그와 동일한 기능이다.

```
[사용방법]
# 대제목
## 중제목
### 소제목
```

## 사용예

# 대제목

## 중제목

### 소제목

# 인용문 태그

blockquote로 > 을 통해 인용문을 작성한다.  
작성예

```
[사용방법]
> 인용문 입력
```

> 인용문1
>
> > 인용문2
> >
> > > 인용문3

# 리스트 목록 (숫자, 번호, 기호)

ol, ul, dl과 동일하다

```
[사용방법]
1. 첫번째
2. 두번째
3. 세번째

* 빨강
    * 주황
        * 파랑
```

1. 첫번째
   - 부제1
   - 부제2
   - 부제3

2. 두번째
   - 목록1
   - 목록2
   - 목록3

# 코드

코드는 백틱문자(`)를 입력하며 이를 동해 자유롭게 코드나 마크다운 문법 등  
필요한 내용을 입력이 가능하다.

````
    [사용방법]

    ```
        안에 들어갈 내용
    ```


````
